PageSpeed Analyzer API Tester

Swagger ReDoc Schema
Not authenticated

Register

POST /auth/register/ — returns access + refresh + user.

Login

POST /auth/login/ — stores the access + refresh tokens.

Current user (me)

GET / PATCH /auth/me/

Password & email

Change password / request reset / verify email.

Analyze a URL

POST /google_speed/analyze/ — async (202). The report starts pending; a worker fills the scores. Poll with “Refresh”.

Reports list

GET /google_speed/reports/ — filter & order.

Report detail / delete

GET / DELETE /google_speed/reports/{id}/ (soft delete).

History

GET /google_speed/history/ — unique URLs + counts (cached 5 min).
